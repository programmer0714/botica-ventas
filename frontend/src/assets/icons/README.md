# 📁 assets/icons — Íconos del Sistema

Coloca aquí todos los íconos SVG del proyecto con los nombres exactos indicados:

| Archivo               | Descripción                          | Usado en         |
|-----------------------|--------------------------------------|------------------|
| favicon.png           | 🖼️ Ícono de la pestaña del navegador | index.html       |
| icon-producto.svg     | 🖼️ Ícono de producto/medicamento     | Ventas           |
| icon-medicamento.svg  | 🖼️ Ícono de medicamento en tabla     | Inventario       |
| icon-reclamo.svg      | 🖼️ Ícono del libro de reclamaciones  | Reclamaciones    |

## Recomendaciones
- Usar íconos SVG de: https://heroicons.com o https://lucide.dev
- Tamaño recomendado: 24x24px viewBox
- Color: usar currentColor para que hereden el color del CSS

## Nota
Si no colocas los íconos, los componentes se ajustan automáticamente.
